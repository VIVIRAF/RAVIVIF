Sejarah Saya

Awal mula saya belajar coding dimulai saat saya masuk jurusan Rekayasa Perangkat Lunak (RPL) di SMK. Tapi seiring berjalannya waktu, ketika saya sudah berada dalam jurusan Rekayasa Perangkat Lunak (RPL) di SMK, minat saya terhadap dunia pemrograman semakin tumbuh. Awalnya, saya memang hanya menganggapnya sebagai main-main belaka. Namun, semakin banyak saya terlibat dalam pelajaran-pelajaran pemrograman, semakin dalam pula rasa tertarik dan terpikat saya terhadap materi tersebut.

Pelajaran demi pelajaran yang saya ikuti dalam bidang pemrograman semakin mengukuhkan minat saya. Saya mulai mempelajari berbagai bahasa pemrograman seperti HTML, CSS, dan JavaScript. Seiring berjalannya waktu, kepuasan yang saya rasakan semakin bertambah ketika saya berhasil menciptakan program-program atau halaman web sederhana.

Seiring perjalanan waktu, minat dan cinta saya terhadap dunia coding semakin berkembang dengan pesat. Saya pun memutuskan untuk mengambil kursus daring dan memanfaatkan berbagai sumber belajar daring agar kemampuan saya terus terasah dan terperinci.

Sejalan dengan peningkatan kemampuan, saya pun mulai mencoba tantangan-tantangan yang lebih kompleks dalam coding. Meskipun perjalanan belajar saya tidak selalu mulus, saya terus gigih dalam menghadapi masalah-masalah yang muncul. Saya belajar untuk mencari solusi dan mengatasi hambatan melalui eksperimen serta penelitian dari berbagai sumber.

Saya merasa sangat bersyukur memiliki komunitas online dan teman-teman sekelas yang mendukung saya dalam perjalanan ini. Kami saling berbagi pengetahuan, pengalaman, dan memberikan bantuan ketika menghadapi kesulitan. Dukungan ini membuat perjalanan belajar saya semakin menyenangkan dan penuh semangat.

Saat ini, saya masih terus belajar dan mengembangkan keterampilan coding saya. Saya sangat yakin bahwa bidang teknologi terus berkembang dengan pesat, dan saya ingin tetap berada di garis depan perkembangannya. Impian saya adalah suatu hari nanti bisa berkontribusi dalam menciptakan solusi-solusi inovatif melalui dunia coding dan pemrograman.
